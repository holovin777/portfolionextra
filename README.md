# PortfolioNextra

This is frontend for PortfolioSpring backend

## Quick Start

```bash
git clone git@github.com:holovin777/portfolionextra.git
cd portfolionextra
yarn dev --hostname 192.168.x.x --port 3000
```

## Build

```bash
yarn build
yarn start --hostname 192.168.x.x --port 4000
```
